\# Actividad Integrada - Semana 2



En esta actividad se trabajó desde una rama secundaria para practicar conflictos y colaboración.



\## Actividades realizadas



\- Clonación del repositorio.

\- Creación de ramas.

\- Realización de commits.

\- Fusión de ramas con git merge.

\- Resolución de conflictos.

\- Invitación de colaborador.

\- Creación de Pull Request.

\- Revisión de código.
